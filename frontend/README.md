# Frontend

Интерфейс **Travel Story** (scrapbook UI) перенесён в **корень репозитория**:

- страницы: `app/story/[tripId]/page.tsx`
- компоненты: `components/travel-story/`, `components/ui/`
- типы: `lib/types/travel-story.ts`
- данные: `GET/ PATCH /api/trip/:id/travel-story`

Эта папка оставлена как якорь в истории git; новый код добавляйте в корневой `app/` и `components/`.

См. основной [README.md](../README.md).
